Patent Portal

Web Programming php,mysql,html,css

* IP (Intellectual Property) Portal is an official institute portal developed by a team 3 students (along with me). This portal implements an interface where a student can upload an abstract of his idea which he wishes to be patented, another interface for moderators to view these abstracts and approve/reject/comment on these abstracts which will then be reflected on the student interface.

* I built the front end and the back end for the moderators interface and wrote code for securing the files uploaded by the students (encrypted using php inbuilt function for TripleDES algorithm).

* It has seen a great overwhelming success over the past year. More than 150 review applications has been submitted and nearly 60 application have been sent for patent filing.

--- As this is a within institute site, if you want to visit the site you can register/login/use the student interface but it's a humble request not to do the final submission

of any project as it may cause further confusion.

Softwares and Frameworks used: HTML, JavaScript, AJAX, jQuery, CSS

  • Made for: Sponsored Research and Industrial Consultancy(SRIC), IIT Kharagpur
  • Date: February 2014
  • Service: Web Development
  • Link: Patent Portal
